Projects

  • Random Link Redirector

    This application will allow you enter several links and will generate a link that contains all those other links. When a user clicks on the generated URL, they will be taken to one of the stored URL's randomly. This was made in JavaScript, using jQuery calls to get and post data.

  • Aversion - a MATLAB game

    A mathematically complex but graphically rudimentary game made entirely on MATLAB. It is a simple game where you control a wave of light and try to dodge asteroids by changing the nature of the wave.

  • Roll the Ball - an Android game

    A visually appealing physics-based game where you attempt to roll a ball to its destination. Complete with scaling graphics and sensitivity depending on user choice. Made with Unreal Engine 4.
